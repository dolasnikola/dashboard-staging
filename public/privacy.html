Privacy Policy

Last updated: March 31, 2026

1. Data Controller

Media House d.o.o., Belgrade, Serbia ("we", "us") operates this dashboard application.

2. What Data We Collect

  • User account data: Email address and name, provided during account setup by an administrator.
  • Advertising campaign data: Aggregated campaign performance metrics (impressions, clicks, spend, conversions) retrieved from third-party ad platforms via authorized API connections.
  • Authentication data: Session tokens managed by Supabase Auth. Passwords are hashed and never stored in plain text.

3. What We Do NOT Collect

  • Personal data of ad viewers or end users
  • Tracking cookies or analytics on dashboard visitors
  • Financial or payment information
  • Location data beyond what ad platforms report in aggregate

4. How We Use Data

All data is used exclusively for:

  • Displaying campaign performance metrics to authorized users
  • Generating PDF performance reports for clients
  • Budget tracking and campaign optimization

5. Data Sharing

We do not sell, rent, or share data with third parties. Campaign data is accessible only to authorized dashboard users assigned to the relevant client account.

6. Data Storage & Security

Data is stored in Supabase (PostgreSQL) with row-level security policies. All connections use HTTPS/TLS encryption. Access is controlled via role-based permissions (admin, account manager, viewer).

7. Third-Party Services

  • Supabase — database and authentication
  • Vercel — application hosting
  • Cloudflare Workers — AI-powered report narrative generation
  • Google Ads API, Meta Marketing API, TikTok Marketing API, Gemius gDE API — campaign data sources

8. Data Retention

Campaign data is retained for as long as the client relationship is active. Users may request account deletion by contacting an administrator.

9. Your Rights

Authorized users may request access to, correction of, or deletion of their personal data by contacting us.

10. Contact

For privacy inquiries: info@mediahouse.rs
